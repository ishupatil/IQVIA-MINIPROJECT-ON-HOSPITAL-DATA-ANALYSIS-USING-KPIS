# Hospital Performance & Patient Analysis Dashboard

A comprehensive healthcare analytics solution built to visualize and analyze hospital performance metrics, patient demographics, and operational trends using interactive dashboards and KPI monitoring.

## üìä Project Overview

This project provides an end-to-end analytics platform for healthcare institutions to track key performance indicators (KPIs), monitor patient trends, analyze payer distributions, and visualize geographical data across multiple dimensions. The solution enables data-driven decision-making for hospital administrators and healthcare professionals.

## üéØ Key Features

### 1. **Hospital Performance & KPI Tracking**
- **Total Entries Monitoring**: Track 13K+ patient entries across the date range
- **Time Series Analysis**: Visualize patient volume and entry trends over time (2017-2022)
- **Encounter Class Distribution**: 
  - Ambulatory (outpatient) services
  - Wellness programs
  - Urgent care visits
  - Emergency services
  - Inpatient admissions
- **Quarterly & Monthly Trends**: Detailed breakdown of entries and total claims by time periods
- **Patient-Level Analytics**: Individual patient entry frequency analysis

### 2. **Payer Network Analysis**
- **Geographic Distribution**: Interactive maps showing payer addresses across the Great Lakes region
- **Location Intelligence**: Visualization of healthcare payer headquarters and service areas
- **Multi-state Coverage**: Analysis spanning Michigan, Wisconsin, Illinois, Indiana, Ohio, Pennsylvania, and New York
- **ZIP Code Mapping**: Detailed location data with color-coded markers

### 3. **Patient Demographics & Trends**
- **Total Patient Count**: Analysis of 974 patients
- **Gender Distribution**:
  - Male Patients: 494 (50.7%)
  - Female Patients: 480 (49.3%)
- **County-Level Analysis**: 
  - Suffolk County leading with 600+ patients
  - Norfolk, Middlesex, Plymouth, and Essex counties analyzed
- **City Distribution**: Detailed breakdown across 15+ cities with Boston representing 55.54% of patients
- **Patient Status Tracking**: 
  - Still Alive: 87 patients (8.9%)
  - Deceased: 2 patients (0.2%)
  - Additional status categories monitored
- **Geographic Visualization**: Interactive maps showing patient distribution by latitude and longitude
